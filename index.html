Minhas Competências

Javascript

Uma linguagem de programação versátil e amplamente adotada, o JavaScript permite que os desenvolvedores criem interatividade e dinamismo em sites. Além de manipular elementos na página, ele também é usado para criar jogos web, aplicativos de página única (SPAs) e interações em tempo real.

HTML

Como a base da web, o HTML ou Linguagem de Marcação de Hipertexto é a linguagem que estrutura o conteúdo de uma página. Usando tags e elementos, os desenvolvedores definem desde parágrafos até imagens e links, possibilitando a criação de uma estrutura coesa e acessível para os navegadores.

CSS

O Cascading Style Sheets (CSS) é a linguagem que traz estilo e estética para as páginas web. Ao definir cores, tipografia, espaçamentos e posicionamentos, os desenvolvedores podem criar interfaces visualmente atraentes e responsivas, melhorando a experiência do usuário.

Figma

Figma é uma poderosa ferramenta de design colaborativo. Com ela, equipes podem criar protótipos interativos e designs de interfaces de usuário de maneira simultânea e em tempo real. Isso facilita a comunicação entre designers e stakeholders, agilizando o processo de design e iterando rapidamente em projetos.

Adobe Photoshop

Uma ferramenta icônica para edição de imagens, o Adobe Photoshop é amplamente utilizado por profissionais de design, fotógrafos e artistas digitais. Com uma ampla gama de recursos, ele permite ajustar cores, retocar fotos e criar composições gráficas complexas, contribuindo para a criação de arte visual impressionante.

DaVinci Resolve

Reconhecido na indústria audiovisual, o DaVinci Resolve oferece uma suíte completa de ferramentas para edição de vídeo, correção de cores, pós-produção de áudio e efeitos visuais. Desde pequenos projetos até grandes produções cinematográficas, ele proporciona um ambiente integrado e profissional para produzir conteúdo visual de alta qualidade.
